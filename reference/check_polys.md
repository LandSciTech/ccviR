# Check and fix polygons

Check and fix polygons

## Usage

``` r
check_polys(poly, var_name = "polygon", quiet = FALSE)
```

## Arguments

- poly:

  Spaitial object. sf or convertible to sf.

- var_name:

  Character. Name of the variable for messaging

- quiet:

  Logical. Whether to suppress progress messages.

## Value

sf polygon
